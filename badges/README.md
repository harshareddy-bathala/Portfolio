# Badges Folder

Place your certification badge images here with the following names:

- `oci-gen-ai.png` - OCI Generative AI Professional badge
- `oci-data-science.png` - OCI Data Science badge
- `ccna.png` - CCNA Enterprise Networking & Security badge

**Recommended image specs:**
- Format: PNG or JPG
- Size: 200x200px to 400x400px
- Transparent background preferred for PNG files

The images will be automatically displayed in the Certifications section. If an image is not found, it will fall back to the icon display.
